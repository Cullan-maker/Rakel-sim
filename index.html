Rakel Lathund

Grundfunktioner

  • På / Av: Håll in Röda luren.
  • Volym: Vrid på toppvredet.
  • Knapplås: Håll in *-knappen.

Ringa Individanrop

  • Slå nummer, välj Stående ikon med pilarna för ISSI (Ring med PTT).
  • Välj Liggande ikon för MSISDN (Ring med Grön lur).
  • Test ISSI: 973221, 973222
  • Test MSISDN: 0701234567, 11414

Spontana RLC-larm

  • Är statusen Fri ropar RLC upp. Svara/Kvittera med PTT. Dubbelklicka Mittval för "Klar, ledig" för att få nya larm.

Direktval & Status

  • Klar, ledig: Dubbelklicka Mittvalsknappen.
  • Se ditt ISSI: Slå *477
  • Mappar: Pil Vänster/Höger. Upp/Ner för bläddring. Tryck OK (Mitt).
  • Larm: Tryck på Röd larmknapp (Toppen).
